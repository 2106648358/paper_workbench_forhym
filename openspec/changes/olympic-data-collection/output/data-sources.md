- "Olympic Games injuries surveillance"

## 6. Google Trends 数据

### 检索方法
1. 访问 https://trends.google.com/
2. 输入项目英文名称 (如 "Athletics Olympic", "Swimming Olympic")
3. 选择时间范围: 过去12个月
4. 记录平均搜索指数

### 示例查询词
- "Athletics Olympic"
- "Swimming Olympic"
- "Gymnastics Olympic"
- "Basketball Olympic"
- "Football Olympic"
