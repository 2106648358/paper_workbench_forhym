# 专家访谈问卷设计

## 奥运项目评估六维指标权重调查问卷

**研究项目**：基于混合权重模型的奥运项目评估与预测系统设计

---

## 一、问卷说明

### 1.1 研究目的

本研究旨在构建一个科学的奥运项目评估体系，通过 AHP-EWM 混合模型确定六维评价指标的权重。您的专业意见对于确定指标的主观权重具有重要价值。

### 1.2 填写说明

本问卷采用 Saaty 的 1-9 标度法，请您对以下六个评估指标进行两两比较，判断其对奥运项目评估的重要性程度。

### 1.3 标度说明

| 标度 | 含义 |
|------|------|
| **1** | 两个指标同等重要 |
| **3** | 前者比后者稍微重要 |
| **5** | 前者比后者明显重要 |
| **7** | 前者比后者强烈重要 |
| **9** | 前者比后者极端重要 |
| **2,4,6,8** | 上述相邻判断的中间值 |
| **倒数** | 若指标 i 与 j 比较为 a，则 j 与 i 比较为 1/a |

---

## 二、六维评估指标说明

| 指标名称 | 英文 | 含义说明 |
|----------|------|----------|
| **流行度** | Popularity | 观众收视率、参与人数、社交媒体讨论度等反映项目受欢迎程度的指标 |
| **性别平等** | Gender Equity | 男女参赛比例均衡度、混合项目数量等反映性别平等的指标 |
| **可持续性** | Sustainability | 碳排放、场馆建设成本、资源消耗等环境影响相关指标 |
| **包容性** | Inclusivity | 残奥项目关联度、发展中国家参与度等反映项目开放程度的指标 |
| **创新性** | Innovation | 项目年轻化程度、技术革新程度、新近加入奥运等反映时代性的指标 |
| **安全性** | Safety | 受伤率、医疗保障需求等反映项目安全风险的指标 |
