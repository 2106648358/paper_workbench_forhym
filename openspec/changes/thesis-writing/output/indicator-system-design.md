## 一、模型整体架构

### 1.1 两层结构设计

```
┌─────────────────────────────────────────────────────────────────┐
│                   两层 AHP-EWM 混合模型                         │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│  目标层：奥运项目综合评分 V                                      │
│      │                                                          │
│      ▼                                                          │
│  ══════════════════════════════════════════════════════════    │
│  第一层：六维指标权重（AHP-EWM 混合）                           │
│  ══════════════════════════════════════════════════════════    │
│      │                                                          │
│      │  ┌─────────────────────────────────────────┐            │
│      │  │ AHP 部分（主观权重）                     │            │
│      │  │ ───────────────────                     │            │
│      │  │ 输入：专家判断矩阵 (6×6)                │            │
│      │  │ 来源：专家访谈                          │            │
│      │  │ 输出：W_AHP = (w₁, w₂, ..., w₆)       │            │
│      │  │ 检验：CR < 0.1                          │            │
│      │  └─────────────────────────────────────────┘            │
│      │                                                          │
│      │  ┌─────────────────────────────────────────┐            │
│      │  │ EWM 部分（客观权重）                     │            │
│      │  │ ───────────────────                     │            │
│      │  │ 输入：六维指标数据矩阵 (n×6)            │            │
│      │  │ 来源：实际收集的数据                    │            │
│      │  │ 输出：W_EWM = (w₁, w₂, ..., w₆)        │            │
│      │  └─────────────────────────────────────────┘            │
│      │                                                          │
│      ▼  混合公式                                                │
│      W = α × W_AHP + (1-α) × W_EWM                             │
│                                                                 │
│      │                                                          │
│      ▼                                                          │
│  ══════════════════════════════════════════════════════════    │
│  第二层：子指标权重（纯 EWM 或等权重）                          │
│  ══════════════════════════════════════════════════════════    │
│      │                                                          │
│      ├─ 流行度 ─── EWM(收视率, 参与人数, 社交媒体)              │
│      ├─ 性别平等 ─ EWM(男女比例, 混合项目)                      │
│      ├─ 可持续性 ─ EWM(碳排放, 成本, 资源) 或等权重             │
│      ├─ 包容性 ─── EWM(残奥关联, 发展中国家参与)               │
│      ├─ 创新性 ─── EWM(年轻化, 技术革新, 新近加入)             │
│      └─ 安全性 ─── 单一指标（受伤率）                           │
│                                                                 │
│      │                                                          │
│      ▼                                                          │
│  项目评分：V_i = Σⱼ wⱼ × Eⱼ(dim_j)                            │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

### 1.2 为什么采用两层结构？

| 层次 | 方法选择 | 数学逻辑 |
|------|----------|----------|
| **第一层** | AHP-EWM 混合 | AHP 捕捉"这个指标应该多重要"（主观价值判断）<br>EWM 捕捉"这个指标实际区分度多大"（客观信息量）<br>混合 = 兼顾价值判断和信息量 |
| **第二层** | 纯 EWM | 子指标层面不需要主观判断<br>让数据说话：哪个子指标更能区分项目？<br>简化模型，避免过度复杂 |

## 二、第一层：六维指标权重

### 2.1 AHP 部分（主观权重）

#### 2.1.1 判断矩阵设计

**判断矩阵结构（6×6）**：

$$A = \begin{bmatrix}
1 & a_{12} & a_{13} & a_{14} & a_{15} & a_{16} \\
1/a_{12} & 1 & a_{23} & a_{24} & a_{25} & a_{26} \\
1/a_{13} & 1/a_{23} & 1 & a_{34} & a_{35} & a_{36} \\
1/a_{14} & 1/a_{24} & 1/a_{34} & 1 & a_{45} & a_{46} \\
1/a_{15} & 1/a_{25} & 1/a_{35} & 1/a_{45} & 1 & a_{56} \\
1/a_{16} & 1/a_{26} & 1/a_{36} & 1/a_{46} & 1/a_{56} & 1
\end{bmatrix}$$

#### 2.1.2 专家访谈设计

**专家要求**：
- 人数：3-5 位
- 背景：体育管理、奥运研究、体育经济学相关领域

**访谈问卷示例**：

```
奥运项目评估六维指标重要性比较

请使用 1-9 标度对以下指标进行两两比较：

标度说明：
1 = 同等重要
3 = 稍微重要
5 = 明显重要
7 = 强烈重要
9 = 极端重要
2,4,6,8 = 中间值

比较问题：
1. 对于奥运项目评估，"流行度"与"性别平等"哪个更重要？重要多少倍？
2. 对于奥运项目评估，"流行度"与"可持续性"哪个更重要？重要多少倍？
...（共 15 个比较问题）
```

#### 2.1.3 多专家判断矩阵整合

当有 k 位专家时，采用几何平均法整合：

$$\bar{a}_{ij} = \left( \prod_{l=1}^{k} a_{ij}^{(l)} \right)^{1/k}$$

#### 2.1.4 一致性检验

计算一致性比率：
$$CR = \frac{CI}{RI} = \frac{\lambda_{\max} - n}{(n-1) \times RI}$$

要求 CR < 0.1，否则需要重新访谈调整。

### 2.2 EWM 部分（客观权重）

#### 2.2.1 输入数据

六维指标数据矩阵：
$$X^{(1)} = \begin{bmatrix}
E_{11} & E_{12} & E_{13} & E_{14} & E_{15} & E_{16} \\
E_{21} & E_{22} & E_{23} & E_{24} & E_{25} & E_{26} \\
\vdots & \vdots & \vdots & \vdots & \vdots & \vdots \\
E_{n1} & E_{n2} & E_{n3} & E_{n4} & E_{n5} & E_{n6}
\end{bmatrix}$$

其中 $E_{ij}$ 是项目 i 在维度 j 上的综合值（由第二层计算得出）。

#### 2.2.2 EWM 权重计算

1. 标准化：$\tilde{E}_{ij} = \frac{E_{ij} - \min(E_j)}{\max(E_j) - \min(E_j)}$

2. 计算比重：$p_{ij} = \frac{\tilde{E}_{ij}}{\sum_{i=1}^{n} \tilde{E}_{ij}}$

3. 计算信息熵：$H_j = -\frac{1}{\ln n} \sum_{i=1}^{n} p_{ij} \ln p_{ij}$

4. 计算熵权：$w_j^{EWM} = \frac{1 - H_j}{\sum_{j=1}^{6} (1 - H_j)}$

### 2.3 混合权重

$$W = \alpha \cdot W_{AHP} + (1-\alpha) \cdot W_{EWM}$$

其中 $\alpha$ 的确定方法：
- 灵敏度分析：测试 α = 0.1, 0.2, ..., 0.9 的结果稳定性
- 推荐：α = 0.5（等权重混合）

## 三、第二层：子指标权重

### 3.1 各维度子指标设计

#### 3.1.1 流行度（Popularity）

**子指标**：
- $x_1$：收视率（TV Viewership）
- $x_2$：参与人数（Participation）
- $x_3$：社交媒体讨论度（Social Media Engagement）

**计算方法**：
$$E_{pop} = \sum_{k=1}^{3} w_k^{(pop)} \cdot \tilde{x}_k$$

其中 $w_k^{(pop)}$ 由 EWM 从实际数据计算得出。

**[待办] 数据来源**：
- 收视率：IOC 报告、Nielsen
- 参与人数：国际运动联合会 (IFs) 统计
- 社交媒体：Twitter/X API、Instagram API

#### 3.1.2 性别平等（Gender Equity）

**子指标**：
- $x_1$：男女参赛比例均衡度
- $x_2$：混合项目数量占比

**计算方法**：
$$E_{gender} = w_1^{(gender)} \cdot (1 - |P_{male} - 0.5| \times 2) + w_2^{(gender)} \cdot \frac{N_{mixed}}{N_{total}}$$

**[有数据] 数据来源**：IOC 官方报告（公开数据）

#### 3.1.3 可持续性（Sustainability）

**子指标**：
- $x_1$：碳排放量（反向）
- $x_2$：场馆建设成本（反向）
- $x_3$：资源消耗（反向）

**计算方法**：
$$E_{sust} = \sum_{k=1}^{3} w_k^{(sust)} \cdot \tilde{x}_k$$

**注意**：此为负向指标，需反向标准化。

**[待办] 数据来源**：
- 碳排放：IOC 可持续发展报告（可能需要估算）
- 场馆成本：奥运筹备报告
- 资源消耗：IFs 环境报告

**备选方案**：若数据难以获取，采用等权重 $w_k = 1/3$。

#### 3.1.4 包容性（Inclusivity）

**子指标**：
- $x_1$：残奥项目关联度
- $x_2$：发展中国家参与度

**计算方法**：
$$E_{incl} = w_1^{(incl)} \cdot D_{para} + w_2^{(incl)} \cdot D_{dev}$$

其中：
- $D_{para}$：残奥关联度（项目是否在残奥会存在）
- $D_{dev}$：发展中国家参与比例

**[有数据] 数据来源**：
- 残奥关联：IPC（国际残奥委会）项目列表
- 发展中国家参与：参赛国统计数据

#### 3.1.5 创新性（Innovation）

**子指标**：
- $x_1$：年轻化程度（青年参与者比例）
- $x_2$：技术革新程度
- $x_3$：新近加入程度

**计算方法**：
$$E_{innov} = \sum_{k=1}^{3} w_k^{(innov)} \cdot \tilde{x}_k$$

**[待办] 数据来源**：
- 年龄分布：IFs 统计（需调研）
- 技术革新：专家评估或文献分析
- 加入年份：Olympedia（公开历史数据）

#### 3.1.6 安全性（Safety）

**单一指标**：受伤率（反向）

**计算方法**：
$$E_{safety} = 1 - \tilde{I}$$

其中 $I$ 为受伤率（每1000小时受伤事件数）。

**[待办] 数据来源**：IOC 医疗报告、学术研究文献

## 四、完整计算流程

### 4.1 算法流程图

```
输入：专家判断矩阵 A，子指标数据矩阵 X

步骤 1：第一层 AHP 权重
    ├── 计算 A 的最大特征值 λ_max
    ├── 求特征向量并归一化 → W_AHP
    └── 一致性检验 CR < 0.1

步骤 2：第二层子指标 EWM 权重
    ├── 对每个维度的子指标数据标准化
    ├── 计算 EWM 权重 w_k^{(dim)}
    └── 计算各维度综合值 E_dim

步骤 3：第一层 EWM 权重
    ├── 构建六维指标矩阵 E = [E_ij]
    ├── 标准化并计算信息熵
    └── 计算 EWM 权重 W_EWM

步骤 4：混合权重
    ├── 选择 α 值（建议 0.5）
    └── W = α × W_AHP + (1-α) × W_EWM

步骤 5：项目综合评分
    └── V_i = Σⱼ wⱼ × E_ij

输出：项目综合评分向量 V，项目排序
```
