## Why

毕业论文题目为**《基于混合权重模型的奥运项目评估与预测系统设计》**，核心目标是**构建并验证一个科学有效的奥运项目评估模型**。论文以数学建模为主线，从问题建模到理论证明，实验验证与系统实现为支撑。

## What Changes

基于 goal.md 的四项研究要求：

### 1. 六维指标评价体系构建
- 包含流行度、性别平等、可持续性、包容性、创新性、安全性六维指标
- 各指标需明确量化公式和数据来源

### 2. AHP-EWM 混合权重计算模块
- 实现 AHP 层次分析法主观权重计算
- 实现熵权法客观权重计算
- 设计混合权重组合策略
- 开发可视化评分系统（计算结果展示）

### 3. 历史数据验证与预测
- 收集历史奥运项目数据
- 验证模型预测准确性
- 对 2032 年奥运项目进行预测

### 4. 系统实现（简化定位）
- 论文核心贡献为**数学模型**，而非系统本身
- 系统定位为**验证工具**，实现计算和可视化功能即可
- Web 应用/仪表盘为扩展目标，可简化为 Python 脚本或 Jupyter Notebook

## Capabilities

### New Capabilities

- **literature-research**: 文献检索与学习工作流，集成 Zotero 文献库，提取方法论和关键结论
- **mathematical-derivation**: 数学建模理论推导，包括 AHP、EWM 的数学原理、公式推导、合理性证明
- **problem-modeling**: 研究问题抽象与建模，将奥运项目评估转化为数学问题
- **experimental-design**: 实验方案设计，包括数据收集、验证方法、对比分析
- **thesis-writing**: LaTeX 论文撰写，章节内容填充，图表编排
- **computation-support**: 计算支持工具，实现权重计算、评分计算、可视化输出
