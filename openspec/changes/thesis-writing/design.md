## Decisions

### 1. 研究问题建模

**问题本质**：奥运项目评估是一个**多准则决策问题**（MCDM）

```
┌─────────────────────────────────────────────────────────────┐
│                     问题抽象                                 │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  现实问题：哪些运动项目应该入选/退出奥运会？                  │
│                                                             │
│       ↓ 转化为                                              │
│                                                             │
│  数学问题：给定 n 个备选项目，m 个评价指标，                  │
│           如何科学计算各项目的综合评分并排序？               │
│                                                             │
│       ↓ 数学表达                                            │
│                                                             │
│  设项目集 S = {s₁, s₂, ..., sₙ}                             │
│  指标集 C = {c₁, c₂, ..., cₘ}                               │
│  决策矩阵 X = [xᵢⱼ]ₙₓₘ，其中 xᵢⱼ 为项目 i 在指标 j 上的值   │
│  目标：求权重向量 W 和综合评分 V = X · W                     │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

### 2. 方法论选择论证

**为什么选择 AHP-EWM 混合模型？**

| 方法 | 优点 | 缺点 | 适用性 |
|------|------|------|--------|
| 纯 AHP | 融入专家经验 | 主观性强 | 适合定性指标多 |
| 纯 EWM | 客观、数据驱动 | 忽略专家知识 | 适合定量指标多 |
| **混合模型** | 兼顾主客观 | 需确定组合系数 | **最适合本研究** |

**数学论证**：
- AHP 捕捉决策者对指标重要性的主观判断
- EWM 反映数据本身的差异性信息
- 混合权重综合利用两方面信息，理论上更完备

### 3. 数学推导框架

#### 3.1 AHP 层次分析法推导

**判断矩阵定义**：
$$A = [a_{ij}]_{n \times n}, \quad a_{ij} > 0, \quad a_{ji} = 1/a_{ij}$$

**权重求解（特征值法）**：
$$AW = \lambda_{max}W$$

**一致性检验**：
$$CI = \frac{\lambda_{max} - n}{n - 1}, \quad CR = \frac{CI}{RI} < 0.1$$

**需在论文中证明/推导**：
1. 为什么特征向量对应权重？
2. 一致性检验的理论依据是什么？
3. CR < 0.1 的来源？

#### 3.2 EWM 熵权法推导

**信息熵定义**：
$$H_j = -\frac{1}{\ln n} \sum_{i=1}^{n} p_{ij} \ln p_{ij}$$

**熵权计算**：
$$w_j = \frac{1 - H_j}{\sum_{j=1}^{m} (1 - H_j)}$$

**需在论文中证明/推导**：
1. 信息熵为什么能衡量指标区分度？
2. 为什么 1-H 代表信息效用值？
3. 熵权与变异系数的关系？

#### 3.3 混合权重组合

**线性组合模型**：
$$W = \alpha W_{AHP} + (1-\alpha) W_{EWM}$$

**组合系数确定方法**：
1. 经验法：取 α = 0.5
2. 灵敏度分析：测试不同 α 的结果稳定性
3. 优化法：基于历史数据拟合最优 α

### 5. 验证实验设计

**验证策略**：
1. **回测验证**：用历史数据检验模型是否能预测已发生的项目变动
2. **对比验证**：与其他方法（纯AHP、纯EWM、TOPSIS）对比
3. **灵敏度分析**：测试参数变化对结果的影响

**数据需求**：
- 2000-2024 年奥运项目清单
- 各项目的六维指标数据
- 已知的项目入选/淘汰决策结果
