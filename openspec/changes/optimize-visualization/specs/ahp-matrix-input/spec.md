## ADDED Requirements

### Requirement: 上三角矩阵数值输入

用户 SHALL 能够在 AHP 判断矩阵的上三角区域直接输入数值。

#### Scenario: 输入数值后自动填充倒数
- **WHEN** 用户在判断矩阵的 上三角单元格输入数值 n
- **THEN** 系统自动在下三角对称位置填充 1/n

#### Scenario: 输入验证
- **WHEN** 用户输入非正数或非数字
- **THEN** 系统显示错误提示，拒绝输入

### Requirement: 判断矩阵编辑模式

系统 SHALL 提供判断矩阵的编辑模式切换功能。

#### Scenario: 切换到编辑模式
- **WHEN** 用户点击"编辑矩阵"按钮
- **THEN** 上三角单元格变为可编辑输入框
- **AND** 显示"保存"和"取消"按钮

#### Scenario: 保存编辑结果
- **WHEN** 用户修改数值后点击"保存"
- **THEN** 系统重新计算权重和一致性检验
- **AND** 显示更新后的结果
