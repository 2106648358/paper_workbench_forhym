## 二、判断矩阵整合方法

### 2.1 几何平均法

对于 k 位专家的判断矩阵，整合方法为：

$$\bar{a}_{ij} = \left( \prod_{l=1}^{k} a_{ij}^{(l)} \right)^{1/k}$$

### 2.2 整合后判断矩阵

|  | 流行度 | 性别平等 | 可持续性 | 包容性 | 创新性 | 安全性 |
|---|--------|----------|----------|--------|--------|--------|
| **流行度** | 1 | ā₁₂ | ā₁₃ | ā₁₄ | ā₁₅ | ā₁₆ |
| **性别平等** | 1/ā₁₂ | 1 | ā₂₃ | ā₂₄ | ā₂₅ | ā₂₆ |
| **可持续性** | 1/ā₁₃ | 1/ā₂₃ | 1 | ā₃₄ | ā₃₅ | ā₃₆ |
| **包容性** | 1/ā₁₄ | 1/ā₂₄ | 1/ā₃₄ | 1 | ā₄₅ | ā₄₆ |
| **创新性** | 1/ā₁₅ | 1/ā₂₅ | 1/ā₃₅ | 1/ā₄₅ | 1 | ā₅₆ |

## 三、Python 计算代码

```python
import numpy as np

def geometric_mean(matrices):
    """
    计算多个判断矩阵的几何平均
    
    Parameters:
    matrices: list of numpy arrays, 判断矩阵列表
    
    Returns:
    整合后的判断矩阵
    """
    k = len(matrices)
    result = np.ones_like(matrices[0])
    for m in matrices:
        result = result * m
    return np.power(result, 1/k)

def calculate_weights(matrix):
    """
    计算AHP权重
    
    Parameters:
    matrix: numpy array, 判断矩阵
    
    Returns:
    权重向量, 最大特征值, CR值
    """
    n = matrix.shape[0]
    
    # 计算特征值和特征向量
    eigenvalues, eigenvectors = np.linalg.eig(matrix)
    
    # 最大特征值
    max_idx = np.argmax(eigenvalues.real)
    lambda_max = eigenvalues[max_idx].real
    
    # 权重向量
    weights = eigenvectors[:, max_idx].real
    weights = weights / weights.sum()  # 归一化
    
    # 一致性检验
    CI = (lambda_max - n) / (n - 1)
    RI = {3: 0.52, 4: 0.89, 5: 1.12, 6: 1.26, 7: 1.36, 8: 1.41, 9: 1.46}
    CR = CI / RI[n]
    
    return weights, lambda_max, CR

# 示例使用
if __name__ == "__main__":
    # 示例判断矩阵（需要替换为实际收集的数据）
    A1 = np.array([
        [1, 2, 3, 4, 5, 6],
        [1/2, 1, 2, 3, 4, 5],
        [1/3, 1/2, 1, 2, 3, 4],
        [1/4, 1/3, 1/2, 1, 2, 3],
        [1/5, 1/4, 1/3, 1/2, 1, 2],
        [1/6, 1/5, 1/4, 1/3, 1/2, 1]
    ])
    
    # 计算权重
    weights, lambda_max, CR = calculate_weights(A1)
    
    print("权重向量:", weights)
    print("最大特征值:", lambda_max)
    print("CR值:", CR)
    print("一致性检验:", "通过" if CR < 0.1 else "未通过")
```

## 五、注意事项

1. **一致性检验**：每位专家的判断矩阵都需要进行一致性检验（CR < 0.1），不通过的需重新访谈

2. **数据保存**：保留每位专家的原始判断矩阵，便于后续分析

3. **异议处理**：如某位专家判断与其他专家差异较大，可考虑：
   - 与该专家沟通确认
   - 在论文中说明情况
   - 考虑剔除异常数据
