## Context

论文需要真实数据支撑实验验证。当前状态：
- `tex_relation/thesis/data/olympic_projects_data.json` 是模拟数据
- 六维指标数据均无真实来源
- 需要系统性收集可追溯的真实数据

**约束条件**：
- 时间约束：数据收集需在论文完成前完成
- 可获取性约束：部分数据难以直接获取，需要代理变量
- 学术约束：所有数据来源需可引用、可追溯

## Goals / Non-Goals

**Goals:**
- 为六维指标收集真实、可引用的数据
- 建立15-20个奥运项目的完整数据集
- 记录每个数据的来源、获取日期、处理方法
- 更新论文第四章的数据来源说明

**Non-Goals:**
- 不追求所有指标都有完美数据（接受代理变量）
- 不开发实时数据采集系统
- 不收集所有历史奥运项目的数据（聚焦于代表性项目）

## Decisions

### 1. 数据收集策略：分层获取

**决策**：按可获取性分层收集数据，优先完成高可获取性数据

```
P0 (必须获取):
├── 性别平等数据 → IOC 官方报告、Olympedia
├── 新近加入年份 → Olympedia
└── 残奥关联 → IPC 官网

P1 (尽量获取):
├── 参与人数 → 各 IFs 官网统计
└── 发展中国家参与 → 奥运参赛国统计

P2 (代理变量):
├── 受伤率 → 学术文献检索
└── 社交媒体热度 → Google Trends 指数

P3 (设计代理):
└── 可持续性 → 代理变量设计
```

**理由**：分优先级可确保核心数据真实，同时控制时间成本

### 2. 项目选择：代表性项目

**决策**：选择15-20个代表性奥运项目，覆盖不同类型

```
核心项目 (Core):
├── 田径、游泳、体操 (传统项目)
├── 篮球、足球 (热门球类)
└── 乒乓球、羽毛球 (技巧类)

新增项目 (New):
├── 攀岩、冲浪、滑板 (2020新增)
└── 霹雳舞 (2024新增)

候选项目 (Candidate):
├── 电子竞技、匹克球
└── 板球、壁球

移除项目 (Removed):
├── 空手道 (2024移除)
└── 棒球/垒球 (历史变迁)
```

**理由**：覆盖不同类型项目可验证模型的普适性

### 3. 数据来源设计：每个指标的获取方案

#### 3.1 性别平等 (Gender Equity)

```
数据来源:
┌─────────────────────────────────────────────────────────────┐
│ 子指标              │ 来源              │ URL/参考文献      │
├─────────────────────────────────────────────────────────────┤
│ 男女参赛比例        │ IOC Official Report │ olympics.com    │
│ 混合项目数量        │ Olympedia         │ olympedia.org   │
└─────────────────────────────────────────────────────────────┘

计算公式:
E_gender = w1 × (1 - |P_male - 0.5| × 2) + w2 × (N_mixed / N_total)

数据获取步骤:
1. 访问 IOC 官网 → 每届奥运会报告
2. 提取各项目男女运动员数量
3. 计算男女比例均衡度
4. 统计混合项目数量
```

#### 3.2 流行度 (Popularity)

```
数据来源:
┌─────────────────────────────────────────────────────────────┐
│ 子指标              │ 来源              │ 可获取性        │
├─────────────────────────────────────────────────────────────┤
│ 全球参与人数        │ 各 IFs 官网       │ ⚠️ 中等         │
│ Google搜索热度      │ Google Trends     │ ✅ 高           │
│ 加入年份            │ Olympedia         │ ✅ 高           │
└─────────────────────────────────────────────────────────────┘

代理变量设计:
- 使用 Google Trends 平均搜索指数替代收视率
- 参与人数可从 IFs 年度报告获取

计算公式:
E_pop = w1 × Norm(Participation) + w2 × Norm(GoogleTrends)
```

#### 3.3 可持续性 (Sustainability)

```
问题: 原始数据 (碳排放、场馆成本) 几乎无法获取

代理变量设计:
┌─────────────────────────────────────────────────────────────┐
│ 代理变量            │ 定义              │ 数据来源        │
├─────────────────────────────────────────────────────────────┤
│ 是否需要专用场馆    │ 1=需要, 0=通用    │ 项目特性分析    │
│ 场馆建设复杂度      │ 1-5评分          │ 文献/专家判断   │
│ 设备依赖程度        │ 1-5评分          │ 项目特性分析    │
└─────────────────────────────────────────────────────────────┘

计算公式:
E_sust = 1 - (VenueScore + EquipmentScore) / 10

说明: 可持续性分数越高，表示越可持续
```

#### 3.4 包容性 (Inclusivity)

```
数据来源:
┌─────────────────────────────────────────────────────────────┐
│ 子指标              │ 来源              │ 可获取性        │
├─────────────────────────────────────────────────────────────┤
│ 残奥关联            │ IPC 项目列表      │ ✅ 高           │
│ 发展中国家参与度    │ 奥运参赛统计      │ ⚠️ 中等         │
└─────────────────────────────────────────────────────────────┘

计算公式:
E_incl = w1 × D_para + w2 × D_dev

其中:
- D_para = 1 (该项目在残奥会存在) / 0 (不存在)
- D_dev = 发展中国家参赛运动员占比

数据获取步骤:
1. 访问 IPC 官网 → Paralympic Sports 列表
2. 对比奥运项目，标记残奥关联
3. 从 Olympedia 提取参赛国数据
4. 计算发展中国家参与比例
```

#### 3.5 创新性 (Innovation)

```
数据来源:
┌─────────────────────────────────────────────────────────────┐
│ 子指标              │ 来源              │ 可获取性        │
├─────────────────────────────────────────────────────────────┤
│ 加入奥运年份        │ Olympedia         │ ✅ 高           │
│ 是否为青年奥运项目  │ YOG 官网          │ ✅ 高           │
│ 技术革新程度        │ 文献/专家判断     │ ⚠️ 中等         │
└─────────────────────────────────────────────────────────────┘

代理变量设计:
- 新近程度 = f(加入年份, 距今年数)
- 年轻化 = 是否为青年奥运项目 × 权重

计算公式:
E_innov = w1 × RecencyScore + w2 × YouthScore + w3 × TechScore
```

#### 3.6 安全性 (Safety)

```
数据来源:
┌─────────────────────────────────────────────────────────────┐
│ 子指标              │ 来源              │ 可获取性        │
├─────────────────────────────────────────────────────────────┤
│ 运动受伤率          │ 学术文献          │ ⚠️ 中等         │
│ 是否对抗性项目      │ 项目特性          │ ✅ 高           │
└─────────────────────────────────────────────────────────────┘

文献来源示例:
- Junge et al. (2009). Injuries in team sport tournaments. Br J Sports Med.
- Engebretsen et al. (2013). Sports injuries and illnesses during the London 2012 Olympic Games. Br J Sports Med.

计算公式:
E_safety = 1 - Norm(InjuryRate)  # 受伤率越高，安全性越低

代理变量:
- 若无受伤率文献，使用对抗性程度评分 (足球=高对抗，体操=低对抗)
```

## Risks / Trade-offs

| 风险 | 影响 | 缓解措施 |
|------|------|----------|
| 部分数据无法获取 | 指标不完整 | 使用代理变量替代 |
| 数据年份不一致 | 对比分析困难 | 统一使用最近完整数据年份 |
| 代理变量主观性 | 结果可能受质疑 | 明确标注代理变量及其局限性 |
| 时间成本超预期 | 影响论文进度 | 优先完成 P0 数据，P3 可简化 |

## Migration Plan

1. **Phase 1**: 收集 P0 数据 (性别平等、残奥关联、加入年份)
2. **Phase 2**: 收集 P1 数据 (参与人数、发展中国家参与)
3. **Phase 3**: 收集 P2 数据并设计代理变量
4. **Phase 4**: 整合数据，更新论文
